# AGENTS.md

This file describes data to be used by AI agents when working on this app, following the [agents.md specification](https://agents.md/)

## App structure

- **docs/** contains the HTML site used to render in GitHub Pages

## Agent capabilities

This agent is designed to:

1. Build HTML site in docs/ to run GitHub Pages
2. Agent is allowed to use only HTML, CSS, and JavaScript inside docs/
3. Agent is used through [cline CLI](https://github.com/cline/cline)
4. Automatically commit the changes after updating files (Git Header)

## Git Header

AI steps to commit changes

1. git add .
2. git commit -m ''
3. git push
